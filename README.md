# e-studies-app

Java desktop application for managing study-related tasks.

## Prerequisites
- Java 17+
- Maven 3.8+

## Build & Run

```bash
# Package application
mvn clean package

# Run application
java -cp target/e-studies-app-1.0.0.jar \
  gr.panagiotisbellias.e.studies.app.EStudiesApp
```

## Development

- Source code: `src/main/java`
- Tests: `src/test/java`
